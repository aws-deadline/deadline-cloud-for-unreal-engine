# UnrealDeadlineCloudService

- Provides a UI and API for submitting render jobs on the artist machine.\
- Executes rendering on Worker Node 


## Requirements
- Unreal Engine 5.2


## Installation
Should be placed in the "Plugins" folder of the UE project.
